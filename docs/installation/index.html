Installation

Installing and configuring Katan is very easy, unlike many panels nowadays that you spend more time installing than using it, formats the machine dozens of times due to errors in the installation, here is different.

System Requirements#

In order to install the Katan server you will need to have the following packages:

PackageDescription
Java
https://www.java.com		Katan was developed using the Kotlin programming language that compiles for the Java Virtual Machine (JVM). So, you need to have Java installed.
Docker
https://www.docker.com		Docker is a set of platform-as-a-service products that use operating system-level virtualization to deliver software in packages called containers.

Downloading#

You can choose the latest version of Katan or a stable version of it in the releases tab.
