## Description
Stops a running server.

## Usage
```console
katan server stop [OPTIONS] SERVER NAME [COMMAND] [ARG...]
```

## Options
| Name     | Default value | Description                                                       |
| :------- | :------------ | :---------------------------------------------------------------- |
| -t       | 10  | Timeout **in seconds** until killing the server if it is not stopped. |
