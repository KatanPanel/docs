## Description
Registers a new account.

## Usage
```console
katan account register [OPTIONS] [COMMAND] [ARG...]
```

## Options
| Name           | Default value | Description                        |
| :------------- | :------------ | :--------------------------------- |
| --username, -u |               | Account username (max: 32 length). |
| --password, -p | `""`          | Account password.                  |
