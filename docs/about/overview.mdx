Katan is a modern, fast and simple game server management panel built run game servers in isolated
environment that can run on multiple platforms.

Created with the aim of facilitating the creation of game servers quickly and effectively.
Proceed to the installation tutorial to get started.

## Open source initiative
Katan is completely open-source and free to install and use. All of our applications are also open source and we plan to maintain that.

We greatly appreciate those who are already there and those who are willing to contribute to the growth and improvement of the Katan, everyone can contribute.

If you want to become a contributor, see [other of our projects](https://github.com/KatanPanel).
